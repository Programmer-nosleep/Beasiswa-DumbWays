# Mini Proyek Portofolio

Sebuah mini proyek portofolio pribadi yang dirancang untuk menampilkan perkenalan sederhana dan kartu profil.

## Deskripsi

Proyek ini adalah halaman web responsif sederhana yang dibangun dengan HTML dan CSS. Halaman ini menampilkan tata letak kartu yang bersih dan terpusat untuk memperkenalkan pengguna, menyediakan opsi kontak, tautan media sosial, ve menampilkan foto profil.

Desainnya minimalis dan modern, menggunakan jenis huruf "Poppins" untuk gaya tipografi yang bersih.

## Teknologi yang Digunakan

- **HTML5**: Untuk struktur halaman web.
- **CSS3**: Untuk gaya (styling), termasuk Flexbox untuk tata letak dan pemusatan elemen.
- **Font Awesome**: Untuk ikon media sosial dan tombol aksi.
- **Google Fonts**: Menggunakan keluarga font "Poppins".

## Instalasi & Penggunaan

Untuk melihat proyek ini, cukup buka file `index.html` di browser web modern apa pun.

1.  Clone atau unduh repositori ini.
2.  Buka folder proyek.
3.  Klik dua kali `index.html` untuk membukanya.

## Fitur

- **Desain Responsif**: Tata letak menyesuaikan untuk layar yang lebih kecil menggunakan Media Queries.
- **Kartu Profil**: Menampilkan gambar pengguna, nama, dan peran/pekerjaan.
- **Tautan Sosial**: Ikon interaktif untuk LinkedIn, Instagram, Facebook, dan Twitter.
- **Call to Action**: Tombol untuk tindakan utama (Hubungi, Unduh CV).

## Aset

- Gambar disimpan di dalam direktori `assets/`.
